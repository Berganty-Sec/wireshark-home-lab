# Project: Wireshark Home Lab Investigation

## Objective

Capture and analyze network traffic to identify:

* Accessed websites
* Network protocols in use
* External communication endpoints

## Environment

* Windows 10
* Wireshark
* Home network

## Results

* Accessed domains: google.com, youtube.com, github.com
* Observed protocols: DNS, TCP, TLS
* Detected communication with multiple external servers
